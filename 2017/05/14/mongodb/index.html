创建

  1. 创建数据库/切换数据库 use [数据库名]

增

  1. 新增参数 

    db.[表名].insert({key: value})

  2. 新增索引

    db.[表名].createIndex({key:value})

删

  1. 删除表 

    db.[表名].drop()

  2. 删除指定参数(默认所有记录) 

    db.[表名]. remove({key:value})

  3. 删除索引 

    db.[表名].dropIndex({key:value}) 
db.[表名].dropIndex(索引名)

改

db.[表名].update(<query>,<update>,upsert: <boolean>,multi: <boolean>,writeConcern: <document>)
query : update的查询条件

upsert : 可选，更新不存在的参数时，true为插入，默认是false

multi : 可选，默认是false，只更新找到的第一条记录

writeConcern :可选，抛出异常的级别

  1. 更新指定参数(默认第一条记录) 

    db.[表名].update({key:oldValue}, {key:newValue})

  2. 更新参数部分内容(默认第一条记录) 

    db.[表名].update({key:oldValue}, {$set:{key:newValue}})

db.imooc.insert({x:1,y:2,z:3});

// db.imooc.update({z:3},{y:4}); 
// 这样操作会导致x,z被覆盖，只有{y:4}

db.imooc.update({z:3},{$set:{y:4}})

  3. 更新不存在的参数进行自动创建 

    db.[表名].update({key:oldValue}, {key:newValue}, true)

  4. 更新所有指定参数 

    db.[表名].update({key:newValue}, {key:oldValue}, false, true)

查

  1. 查询数据库中的表 

    show collections

  2. 查询返回一个参数 

    db.[表名].findOne()

  3. 查询所有参数 

    db.[表名].find()

  4. 查询符合标准参数 

    db.[表名].find({key: value})

  5. 查询是否存在参数

    db.[表名].find({key:{$exists:true}})

  6. 查询所有参数个数 

    db.[表名].find().count()

  7. 查询所有参数向后偏移个数 

    db.[表名].find().skip(number)

  8. 查询所有参数返回参数个数

    db.[表名].find().limit(number)

  9. 查询所有参数排序方式 

    db.[表名].find().sort({key:value})

  10. 查询现有索引 

    db.[表名].getIndexes()
索引类型

  1. _id索引

  2. 单建索引

    db.[表名].createIndex({x:1})

  3. 多键索引

    // 数据格式{x:[1,2,3,4]}
db.[表名].createIndex({x:1})

  4. 复合索引

    db.[表名].createIndex({x:1,y:1})

  5. 过期索引

    db.[表名].createIndex({x:1}, {expireAfterSeconds: time})

// 数据插入格式（必须为时间戳）
db.[表名].insert({x: new Date()})

  6. 全文索引

    db.[表名].createIndex({key: "text"});

// 数据查询格式
db.[表名].find({$text: {$search: value}})

// value为多个时
// 1. value: "aa bb cc" 包含aa或者bb或者cc
// 2. value: "aa bb -cc" 包含aa或者bb不含cc
// 3. value: " \"aa\" \"bb\" \"cc\" " 包含aa和bb和cc

    • 全文索引相似度查询(配合sort能够很好的进行排序)

      // 数据查询格式
db.[表名].find({$text: {$search:value}}, {score:{$meta:"textScore"}})) 
//.sort({score:{$meta:"textScore"}}))

  7. 地理位置索引

    • 2d索引（平面地理位置索引）

      db.[表名].createIndex({w:"2d"})

// 数据插入格式
// x ∈ [-180,180] y ∈ [-90,90]
db.[表名].insert(w:[x,y])

// 数据查询格式
1. $near查询
    db.[表名].find({w:{$near:[x,y]}}) // 返回最近的100个点
    db.[表名].find({w:{$near:[x,y],$maxDistance:value}})  // 返回距离最近的100个点，最远距离为value

2. $goeWithin查询    
    db.[表名].find({w:{$geoWithin:{$box:[[x1,y1],[x2,y2]]}}}) // 位于矩形状内的点
    db.[表名].find({w:{$geoWithin:{$center:[[x0,y0],r]}}}) // 位于圆形内的点
    db.[表名].find({w:{$geoWithin:{$polygon:[[x1,y1],[x2,y2]...]}}}) // 位于多边形内的点
3. geoNear

    db.runCommand({
        geoNear: [表名],
        near: [x,y],
        maxDistance: v1,
        minDistance: v2, // 对2d索引无效
        num: v3 // 返回数目
    })
    • 2d sphere索引 （球面地理位置索引）

索引属性

db.[表名].createIndex({},{key:value})

  1. 名字 

    db.[表名].createIndex({},{name:value})

  2. 唯一性(可使insert值唯一)

    db.[表名].createIndex({},{unique:true/false})

  3. 稀疏性(默认false，true不必为不存在的字段创建索引)

    // 比如索引为{x:1}
// insert({y:1}) sparse:true 则不创建该索引

db.[表名].createIndex({sparse: true/false})
